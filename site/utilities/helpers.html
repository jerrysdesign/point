Appearance

Class Description
.u-scrollable set a fixed height or width of element and provide a y-axis or x-axis scrollbar
.u-block display: block !important;
u-centerBlock display: block !important;, margin-left: auto !important;, margin-right: auto !important;
.u-inline display: inline !important;
.u-inlineBlock display: inline-block !important;
.u-floatLeft Float an element to the left
.u-floatRight Float an element to the right
.u-clearfix Clear floats
.u-hidden Force an elment to be hidden
.u-hiddenUp@(sm, md, lg) Force an elment to be hidden at breakpoint up
.u-hiddenDown@(sm, md, lg) Force an elment to be hidden at breakpoint down
